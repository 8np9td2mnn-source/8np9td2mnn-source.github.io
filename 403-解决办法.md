# 403 权限错误 — 解决办法

报错含义：GitHub 认出了账号 `8np9td2mnn-source`，但**当前 Token 没有写入仓库的权限**（不是密码错）。

按下面顺序试，**推荐先试「办法一」**。

---

## 办法一：换「经典 Token」（成功率最高）

不要用 Fine-grained，改用 Classic：

1. 打开：https://github.com/settings/tokens  
2. **Generate new token (classic)**  
3. Note 填：`blog-push`  
4. 过期时间选 90 days 或 No expiration  
5. **勾选：`repo`**（整项打勾；一般不需要勾选 `workflow`）  
6. 拉到最下面 **Generate token**  
7. **复制** Token（`ghp_` 开头）

再执行：

```bash
cd ~/personal-blog
./push-with-token.sh
```

粘贴**新的** `ghp_` Token。

---

## 办法二：检查 Fine-grained Token（若坚持用新类型）

打开：https://github.com/settings/tokens?type=beta → 点你创建的 Token

必须同时满足：

| 设置项 | 正确值 |
|--------|--------|
| Resource owner | `8np9td2mnn-source`（你自己） |
| Repository access | **Only select** → 勾选 `8np9td2mnn-source.github.io` |
| Contents | **Read and write** |
| Metadata | **Read-only**（默认有） |

改完后点 **Save**，重新运行 `./push-with-token.sh`。

---

## 办法三：不用 Token — 网页上传（一定可用）

用浏览器登录 GitHub，走网页提交，不经过 git push。

### 步骤

1. 打开：https://github.com/8np9td2mnn-source/8np9td2mnn-source.github.io  

2. 删除旧主页：  
   - 点 `index.html` → 右上角 **⋯** → **Delete file**  
   - 下方填 `删除旧主页` → **Commit changes**

3. 上传新博客：  
   - 回到仓库首页 → **Add file** → **Upload files**  
   - 打开 Mac 的 **访达** → `个人`（或 home）→ `personal-blog`  
   - 选中下面这些（可 `Cmd+A` 全选后取消勾选 `.git` 文件夹）：  
     - 所有 `.html`、`.css`、`.js`、`posts.json`  
     - `.github` 文件夹  
     - `.gitignore`  
   - **拖到** GitHub 网页上传区  
   - 填说明 `上线个人博客` → **Commit changes**

4. 开启网站：  
   - **Settings** → **Pages**  
   - Source：**Deploy from a branch**  
   - Branch：`main`，文件夹 **`/ (root)`** → **Save**

5. 等 2 分钟，访问：https://8np9td2mnn-source.github.io/  
   按 `Cmd + Shift + R` 强制刷新。

---

## 办法四：GitHub Desktop

1. 安装：https://desktop.github.com/  
2. 用 **8np9td2mnn-source** 账号登录（和网页同一账号）  
3. **File → Add Local Repository** → 选 `personal-blog`  
4. 提交说明填 `上线博客` → **Commit to main**  
5. **Repository → Push**（若提示冲突，用 **Force push**）

---

## 确认是否成功

打开仓库：https://github.com/8np9td2mnn-source/8np9td2mnn-source.github.io  

应能看到很多文件，例如：

- `posts.json`
- `write.html`
- `article.html`
- `.github/workflows/pages.yml`

而不止一个 `index.html`。

---

## 仍报 403？

请确认：

1. 创建 Token 时，浏览器右上角登录的是 **8np9td2mnn-source**（不是公司别的账号）  
2. 粘贴 Token 时**没有多余空格**  
3. 用的是 **Classic `repo` 权限** 的 `ghp_` Token  

把 Token 类型（`ghp_` 还是 `github_pat_`）和用的是办法几告诉我。
