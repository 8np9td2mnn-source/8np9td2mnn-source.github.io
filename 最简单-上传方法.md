# 最简单：把博客传到 GitHub（3 选 1）

**现状：** 你电脑里的博客是对的；GitHub 上仍是旧的「个人主页」，所以网站地址显示不对。  
**目标：** 用下面任意一种方法，把 `personal-blog` 文件夹里的内容传上去。

---

## 方法一：用 Token 推送（推荐，约 5 分钟）

### 1. 创建 Token

1. 打开：https://github.com/settings/tokens?type=beta  
2. **Generate new token**  
3. 仓库只选：`8np9td2mnn-source.github.io`  
4. **Contents** 权限选 **Read and write**  
5. 生成后**复制** Token（`github_pat_` 开头）

### 2. 在终端执行

```bash
cd ~/personal-blog
./push-with-token.sh
```

按提示粘贴 Token（输入时屏幕不显示字符，正常），回车。

看到 `Done` 后，等 2 分钟打开：https://8np9td2mnn-source.github.io/

---

## 方法二：GitHub Desktop（不会用终端）

1. 下载安装：https://desktop.github.com/  
2. 登录你的 GitHub 账号  
3. **File → Add Local Repository** → 选文件夹 `personal-blog`  
4. 若提示创建仓库，点 **create a repository**  
5. **Repository → Repository settings → Remote**  
   地址填：`https://github.com/8np9td2mnn-source/8np9td2mnn-source.github.io.git`  
6. 左下角写摘要 `上线博客`，点 **Commit to main**  
7. 点 **Push origin**（若失败，选 **Force push** 覆盖远程旧文件）  
8. 等 2 分钟刷新网站

---

## 方法三：网页拖放上传（不用 Git）

1. 打开：https://github.com/8np9td2mnn-source/8np9td2mnn-source.github.io  
2. 点进 `index.html` → 右上角 **Delete file** → 提交删除  
3. 回到仓库首页，**Add file → Upload files**  
4. 打开你电脑的 `personal-blog` 文件夹，**选中除 `.git` 以外所有文件和文件夹**拖进网页  
   - 要包含：`index.html`、`posts.json`、`styles.css`、`write.html`、`article.html`、`about.html`、`blog-store.js`、`.github` 文件夹 等  
5. 点 **Commit changes**  
6. 打开 **Settings → Pages**，Source 选 **Deploy from a branch**  
   - Branch: `main`，Folder: `/ (root)`，Save  
7. 等 2 分钟访问网站

---

## 怎么算成功了？

打开 https://8np9td2mnn-source.github.io/ 应看到：

- 左侧「你的名字」+ **文章 / 关于**
- 中间是**文章列表**（不是「找到我」「项目」那种主页）

若还是旧页面：浏览器 **强制刷新**（Mac：`Cmd + Shift + R`）。

---

## 仍不行？

把终端里**红色报错全文**复制发给助手，或说明用的是方法一/二/三哪一步卡住了。
